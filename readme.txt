== Description ==

A lightweight WordPress plugin that protects selected pages and posts behind a custom password page.

= Features =
* Protect specific pages and posts
* Redirect visitors to a custom password page
* AJAX-powered password validation
* Redirect users back to protected content after authentication
* Separate access control for pages and posts
* Password visibility toggle
* WordPress nonce security protection

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the protection settings (Protection Page, Protected Page, Protected Post, and Password) in your dashboard.
4. Save your settings.

== Frequently Asked Questions ==

= How It Works =
1. A user visits a protected page or post.
2. The plugin redirects them to the designated password page.
3. The user enters the correct password.
4. Access is granted.
5. The user is automatically redirected back to the requested content.
